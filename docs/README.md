# 文档索引（docs/）

这里是 GitHub Pages 的静态站点源目录。

## 页面入口

- `docs/index.html`：文档首页（项目定位、阅读顺序、贡献方式）
- `docs/getting-started.html`：快速开始（最短命令链）
- `docs/roadmap.html`：使用与工作流（从用户路径理解 run/tool/diff/approval）
- `docs/architecture.html`：架构与代码导览（对象模型、工具边界、事件、索引、桌面端适配）
- `docs/papers.html`：论文与参考（保持原样）

## 兼容入口

- `docs/codex-alignment.html`：旧入口，跳转到论文页，避免历史链接失效

## 维护原则

- 文档默认只使用中文，避免中英混排导致阅读成本上升。
- 新增后端能力必须在“用户路径”里可被使用：需要桥接与桌面端适配任务同步记录。
